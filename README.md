# CakePHP + fullCalendar = CakeFullCalendar

CakeFC は jQuery のカレンダーアプリを実現するプラグイン
[fullCalendar](http://arshaw.com/fullcalendar/) と
CakePHP を使ってカレンダーアプリを実装する。
CakePHP の fullCalendar 実装を目指す。
